## Troubleshooting

### Issue: "file would be overwritten"

If you see warnings about overwriting files, it's normal - archives are designed to merge.

To suppress warnings:
```bash
unzip -o archive-*.zip  # -o overwrites without prompting
```

### Issue: Different timestamps on duplicate files

Some files appear in multiple archives for convenience (e.g., README files). The versions are identical; timestamp differences don't matter.

### Issue: Missing files after extraction

Ensure you extracted all required archives for your use case:
- Core only: Archive 1
- Testing: Archives 1 + 2
- Complete: Archives 1 + 2 + 3
